The Project

During the spring semester of 2018, I am working with my mentor Dr. Husnu Narman and fellow group member Geanina to develop a tool that uses various API's to analyze an IP addresses reputability. We have the developed the capability for this, however, we are wanting to extend it to be able to use machine learning to identify an IP Address and determine its reputability. The demand for this is high, as its impossible for blacklists to locate every single malicious domain. On this web page, I will be recording a weekly blog of my research as well as posting the resources I have been using to develop this project.

Reports

Week One

During the first week, I began to familiarize myself with the project description as well as begin to learn machine learning. The web application has already been created by my group member using PHP, however, it still needs a lot of work and needs a machine learning implementation.

Week Two

On the second week, I implemented a small python project that utilized supervised learning with logistic regression. It has an excel spreadsheet of over 300,000 URL's and classified them as good or bad. It managed to get an accuracy rating of 96% when using 20% of the data set for training, however, using only URL information is not enough to accurately classify URL's and determine if they are malicious.

Week Three

This week I developed a python project that will run daily API scans and dump the results to a SQL Lite3 database. I also created a serializer for my machine learning implementation so we can begin implementing it on the web application. This is still an extremely rudimentary form of what a good classifier should look at by the end of this project, however, it will give us an idea of the requirements necessary for it to perform on the application.
